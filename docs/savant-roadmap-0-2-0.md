# Savant Roadmap (v0.2.0 → v0.3.0)

## Phase 1 — Foundations
- Implement DB schema for agents, personas, rulesets, workflows, runs
- Build storage layer (Savant::DB)
- Add default AMR + default Tools

## Phase 2 — Agent System
- Agent creation wizard (Persona → Driver → Rules)
- Agent list page
- Agent detail page (editable configs)
- Agent run viewer (chat-like transcript)
- CLI alignment (agent create/run/list)

## Phase 3 — Workflow System
- Workflow creation wizard
- Visual graph editor
- Workflow list page
- Workflow detail page
- Workflow run viewer (chat-style)
- THINK → PLAN → NEXT integration
- Workflow DB storage

## Phase 4 — Runtime Experience
- Unified transcript renderer
- Streaming execution
- Error recovery & restart

## Phase 5 — Polish & Stability
- Favorites
- Sorting & filters
- Metadata enhancements
- UX consistency improvements
